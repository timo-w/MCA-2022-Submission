    Week 5

    Create Metadata Schema

    Create a metadata schema by listing the elements (or a subset of at least 5 elements) that your group chose back on Tuesday.

    Chosen Elements:

    • Title
    • Author/Composer
    • Year Produced/Composed
    • Key
    • Tempo
    • Encoder
    • Transcriber

    Add Metadata to MEI File

    Modify your MEI document according to the schema that you have created. Where possible, enrich your data by linking it to existing authorities as discussed in yesterday"s lecture using the @auth.uri attribute.
